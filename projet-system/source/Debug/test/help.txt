SYNOPSIS
     tar [-v] [-z] [-f [outfile.tar]] -c files ...
     tar [-v] -t file.tar
     tar [-v] -x file.tar [pathToExtract]
     tar [-v] [-u] -r file.tar filesToAdd ...
     tar [-v] -d file.tar fileToRemove

DESCRIPTION
     Tar regroupe des fichiers et dossiers dans une seul archive

FUNCTION LETTERS
     Main operation mode:

     -h, afficher l'aide
     
     -v, rend le programme verbeux

     -c, créer une nouvelle archive à partir d'une liste de fichier

     -d, nd differences between archive and file system

     --delete
           delete from the archive (not on mag tapes!)

     -r, --append
           append files to the end of an archive

     -t, --list
           list the contents of an archive
